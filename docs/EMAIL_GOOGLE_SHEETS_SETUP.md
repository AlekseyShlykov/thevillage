# Как подключить приём почт через Google Таблицу

Игра отправляет на ваш URL POST-запрос с телом `{ "email": "адрес@example.com" }`. Ниже — два способа принять эти данные.

---

## Вариант 1: Google Таблица + Apps Script

Письма будут попадать в строки Google Таблицы.

### Шаг 1. Создайте таблицу

1. Откройте [Google Таблицы](https://sheets.google.com) и создайте новую таблицу (например, «Winter Village — почты»).
2. В первой строке задайте заголовки: в **A1** — `Email`, в **B1** — `Дата` (по желанию).

### Шаг 2. Откройте редактор скриптов

1. В меню таблицы: **Расширения** → **Apps Script**.
2. Удалите весь код в редакторе и вставьте код ниже.

### Шаг 3. Код скрипта

```javascript
function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var body = e.postData ? JSON.parse(e.postData.contents) : {};
    var email = (body.email || '').toString().trim();
    if (!email) {
      return createResponse(400, { error: 'Email is required' });
    }
    sheet.appendRow([email, new Date()]);
    return createResponse(200, { ok: true });
  } catch (err) {
    return createResponse(500, { error: String(err.message) });
  }
}

function createResponse(code, data) {
  var output = ContentService.createTextOutput(JSON.stringify(data));
  output.setMimeType(ContentService.MimeType.JSON);
  return output;
}
```

Сохраните проект (Ctrl+S), название можно оставить любым (например, «EmailReceiver»).

### Шаг 4. Разверните как веб‑приложение

1. Нажмите **Развернуть** → **Новое развёртывание**.
2. Рядом с «Выберите тип» нажмите **Настройки** (шестерёнка) → выберите **Веб‑приложение**.
3. Укажите:
   - **Описание**: например, «Приём почт из игры».
   - **Запуск от имени**: **Я** (ваш аккаунт).
   - **У кого есть доступ**: **Все пользователи** (чтобы игра могла слать запросы без входа).
4. Нажмите **Развернуть**.
5. В окне «Развёртывание» нажмите **Копировать** у поля **URL веб‑приложения** — это и есть ваш адрес для игры.

### Шаг 5. Подключите URL к игре

Откройте в проекте файл `public/data/game-balance.json` и вставьте скопированный URL:

```json
"demoEnd": {
  "emailSubmitUrl": "https://script.google.com/macros/s/ВАШ_ID/exec"
}
```

Замените `https://script.google.com/macros/s/ВАШ_ID/exec` на ваш скопированный URL целиком.

### Важно про CORS

Если игра открыта с другого домена (например, с GitHub Pages или itch.io), браузер может заблокировать запрос к Google из‑за CORS. В таком случае:

- либо используйте **Вариант 2 (Formspree)** ниже — он нормально работает с любых сайтов;
- либо проверьте в консоли браузера (F12 → вкладка «Сеть» / Network), уходит ли POST и какой приходит ответ.

## Вариант 2: Formspree (проще и без CORS)

[Formspree](https://formspree.io) даёт готовый URL для приёма данных и корректно отвечает с любых доменов.

### Шаг 1. Регистрация и форма

1. Зайдите на [formspree.io](https://formspree.io) и зарегистрируйтесь (есть бесплатный план).
2. Нажмите **New Form**.
3. В качестве имени поля укажите **email** (именно так, без заглавных).
4. Сохраните форму — Formspree покажет URL вида:  
   `https://formspree.io/f/xxxxxxxx`

### Шаг 2. Подключите URL к игре

В `public/data/game-balance.json` укажите этот URL:

```json
"demoEnd": {
  "emailSubmitUrl": "https://formspree.io/f/xxxxxxxx"
}
```

Подставьте свой идентификатор вместо `xxxxxxxx`. Игра уже отправляет JSON `{ "email": "..." }` — Formspree такое тело принимает и записывает в поле `email`. Письма будут приходить на почту, привязанную к Formspree, и (при желании) их можно экспортировать или связать с Google Таблицей через интеграции Formspree.

---

## Проверка

1. Запустите игру (локально или на хостинге).
2. Нажмите кнопку **✉ Почта** вверху справа, введите тестовый email и нажмите **Отправить**.
3. Для **Варианта 1** — откройте таблицу и проверьте новую строку с почтой и датой.
4. Для **Варианта 2** — проверьте почту или панель Formspree.

Если в игре после отправки показывается «Спасибо!» или «Спасибо! Мы сообщим о выходе полной версии» — запрос прошёл успешно.
